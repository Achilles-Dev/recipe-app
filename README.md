# Recipe App

> The **Recipe app** keeps track of all your recipes, ingredients, and inventory. It allows you to save ingredients, keep track of what you have, create recipes, and generate a shopping list based on what you have and what you are missing from a recipe. Also, since sharing recipes is an important part of cooking, the app allow you to make them public so anyone can access them.


## Built With

- Ruby
- Rails
- Postgresql
- Devise
- CSS & Bootstrap 5
- Rspec
- Capybara

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Ruby
- Rails
- Postgresql


### Setup

- Clone the Repository
`git clone https://github.com/Achilles-Dev/recipe-app.git`

- Go to the project directory
`cd recipe-app`

- Run ```bundle install```
- Run ```rails db:create```
- Run ```rails db:migrate```

### Usage

- Run ```rails s``` to run the app
